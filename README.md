# Dashboard UI

A modern and minimal dashboard user interface built with **React, TypeScript, Vite, and Tailwind CSS**.
This project demonstrates clean UI design suitable for administration or analytics dashboards.

## Features

- Modern dashboard layout with sidebar navigation
- Responsive design for all devices  
- Order management system with sorting and filtering
- Clean and professional UI components
- Built with TypeScript for type safety

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Remix Icons
